Generates finger tables for Chord DHTs  
http://en.wikipedia.org/wiki/Chord_%28peer-to-peer%29  

Each FT line consists of
FT index, value, peer    

Example:  
    ./chordFingerTable -m 7 -peers 1,33,42,80,99,110
